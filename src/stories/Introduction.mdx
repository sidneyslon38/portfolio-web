# NYCity News Service Component Library

<img

This site documents the reusable user-interface components developed for the
[Craig Newmark Graduate School of Journalism](https://www.journalism.cuny.edu/) at The City University of New York. Each component is built with [Svelte 5](https://svelte.dev/) and styled to match the [NYCity News Service](https://www.nycitynewsservice.com/) design system.

It was created for ["JOUR 73361: Coding the News,"](https://palewi.re/docs/coding-the-news/) a course on teaches how America's top news organizations escape rigid publishing systems to design beautiful data-driven stories on deadline. The source code is available on GitHub at
[palewire/cuny-jour-static-site-template](https://github.com/palewire/cuny-jour-static-site-template).
